About These Files

These are real RedVox sample files bundled with the SDK's test suite. They contain synthetic and real sensor data.

API 900 .rdvxz files are the legacy compressed format.

API 1000 .rdvxm files are the modern compressed format, containing richer sensor data.

Workflow
  1. Download a sample file
  2. Upload it to File Inspector to view metadata
  3. Use Signal Analysis for waveform/FFT plots
  4. Use File Converter to change format
  5. Use Validator to check API 1000/M compliance
